🚗 Car Sales Analysis Dashboard

 📊Project Overview

This project presents an interactive Car Sales Analysis Dashboard built using Power BI. It analyzes car sales data to extract meaningful insights related to revenue, sales trends, customer behavior, and product performance.

The dashboard helps stakeholders make data-driven decisions by visualizing key performance indicators (KPIs) and trends in an intuitive way.

---

📁 Dataset

* File Name: car sales data.csv
* The dataset contains detailed information about:

  * Car models
  * Sales transactions
  * Pricing
  * Customer details
  * Sales regions
  * Dates and time-based data

---

🛠️ Tools & Technologies

* Power BI (.pbix) – for dashboard creation and visualization
* Microsoft Excel / CSV – data source
* Data Cleaning & Transformation – Power Query Editor
* DAX (Data Analysis Expressions) – calculated measures and KPIs

---

📌 Key Features

* 📈 Sales Performance Analysis

  * Total Sales Revenue
  * Number of Units Sold
  * Profit Analysis

* 🏆 Top Performing Models

  Best-selling car models
  Revenue contribution by model

* 🌍 Regional Sales Insights

  Sales distribution across different locations
   Region-wise performance comparison

* 📅 Time-Based Trends

  Monthly / Yearly sales trends
  Seasonal demand patterns

* 👥 Customer Insights

  * Customer segmentation
  * Purchase behavior analysis

📊 Dashboard Visuals

The Power BI dashboard includes:

* Bar Charts
* Line Charts
* Pie Charts
* KPI Cards
* Slicers for filtering (Region, Model, Date, etc.)

⚙️ How to Use

1. Download the .pbix file from this repository
2. Open it using Power BI Desktop
3. Ensure the dataset (car sales data.csv) is properly linked
4. Refresh the data if needed
5. Interact with filters and visuals to explore insights
